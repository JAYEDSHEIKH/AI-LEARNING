### Gradient Descent вҖ” Rolling the Ball Downhill

```
  ERROR (loss)
     в”Ӯ
  1.0в”Ө \
     в”Ӯ  \     вҶҗ Starting point (random weights, high error)
  0.7в”Ө   \    вҡҪ
     в”Ӯ    \
  0.4в”Ө     \__
     в”Ӯ        \___
  0.1в”Ө            \____
     в”Ӯ                 \______ вҶҗ Minimum! (best weights)
  0.0в”Ө
     в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв–ә WEIGHTS
       random                         optimal

  Each training step = rolling the ball one small step downhill.
  The "slope" (gradient) tells us which direction to go.
  This process is called GRADIENT DESCENT.
```
